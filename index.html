Synthra

Synthra
AI Courses
Web Design UI / UX Figma AI

Synthra — платформа онлайн-курсов по искусственному интеллекту. Задача дизайна: создать интерфейс, который чувствуется как пространство будущего — технологичный, но не холодный. Тёмная тема с синими и фиолетовыми акцентами подчёркивает инновационность продукта.

Проработан детальный UX: карточки курсов с прогрессом, персональный дашборд, онбординг-воронка и структура навигации. Собран полноценный UI Kit с компонентами, типографикой и цветовой системой для передачи в разработку.
